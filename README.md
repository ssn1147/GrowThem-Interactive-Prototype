GrowThem - Interactive Prototype

Overview:
GrowThem is a mobile application prototype designed to help users identify plants, receive personalized plant care guidance, and connect with a community of plant enthusiasts.

Features:
- Splash Screen
- Login Screen
- Home Screen
- Plant Scanner Screen
- Interactive navigation between screens
- Modern UI designed in Figma

Tools Used:
- Figma
- GitHub
